# 虚幻官方资料

相比于UE4时期缺乏管理的潦草文档，UE5发布后Epic官方在建设文档和社区生态等方面有比较明显的投入和维护。

![UE-5](/images/202511091054626.webp)

## 官方文档

链接：[虚幻引擎官方文档](https://dev.epicgames.com/documentation/zh-cn/unreal-engine/)。

**绝大部分文档都有官方中文翻译**，遇到部分翻译不当或者翻译缺失，给浏览器装一个翻译插件就可以解决。

::: tip 如何切换虚幻引擎文档语言？
点击网页右上角的 🌐 图标，选择对应语言即可。
:::

对于初学者（刚接触游戏开发的人），不推荐完全地将官方文档当做第一个帮助你起步的“教程”，**官方文档其实更像一个纯文本的操作说明**，对于初学者来说，它更适合的定位应该是字典或百科全书。但如果你是从Unity或其他引擎转来的有经验开发者，官方文档会非常适合你。

对于初学者来说，在开始阶段，仅推荐参考以下几个部分：

- [安装虚幻引擎](https://dev.epicgames.com/documentation/zh-cn/unreal-engine/install-unreal-engine)：该文档中比较详实地告诉你该如何安装虚幻引擎，尽管网络上有比较多的教程都会覆盖到这点，但每个版本可能会有些许区别（尤其是大版本），如果碰到问题可以参考对应版本的官方文档；
- [虚幻引擎术语](https://dev.epicgames.com/documentation/zh-cn/unreal-engine/unreal-engine-terminology)：该文档可以帮你快速浏览一遍虚幻引擎中构建玩法相关的概念术语，比如蓝图、对象、类、Actor等，简单理解和记住这些概念，可以帮助你更快理解其他资源和教程，不用刻意理解他们之间的联系，后续实践过程会补全这一部分；
- [目录结构](https://dev.epicgames.com/documentation/zh-cn/unreal-engine/unreal-engine-directory-structure)：选读目录，推荐你安装完引擎、创建第一个项目并成功运行后尝试阅读，提前对项目目录的文件结构有一个认知。

::: tip 链接失效？
虚幻引擎在部分版本更新后可能会调整文档目录，从而导致链接失效，请在 [Github Issues](https://github.com/miko-cn/awesome-ue-guide/issues) 中提醒我们，感谢！
:::

## 官方论坛

![Forums](/images/202511091135066.webp)

链接：[Unreal Engine 论坛](https://forums.unrealengine.com/categories?tag=unreal-engine)

中文社区链接：[Unreal Engine 论坛 / 简体中文](https://forums.unrealengine.com/tags/c/international/simplified-chinese/61/unreal-engine)

在引擎使用过程中碰到一些问题或者报错，如果在谷歌上搜索错误代码，通常会给你一些类似问题的论坛链接。

社区主要是面向英文母语用户，也有一个单独的中文板块，实在没有找到问题解决方案也可以单独开一贴，不懂英文也可以尝试用AI翻译后发帖。

## Epic开发助手

![AI-Assistant](/images/202511091140671.webp)

链接：[开发者助手 - Beta](https://dev.epicgames.com/community/assistant/unreal-engine)

## 官方教程

![Learning](/images/202511091150002.webp)

链接：[学习 Unreal Engine](https://dev.epicgames.com/community/unreal-engine/learning)

截止到本文档发布时，Epic官方已经收录了6000+的课程资源，其中包含Epic官方开发人员推出的课程（1800+），也有社区人员自己编撰的课程（4700+），其中大部分课程都是英文录制，部分包含中文字幕，推荐有一定英语水平的开发者使用。

::: info 推荐看哪些课程？
如果你是初学者，可以找一些[入门级的课程](https://dev.epicgames.com/community/unreal-engine/learning/page/2?is_beginner=true)，教程目录会有筛选器，可以选择“适用于初学者”。

具体的课程推荐限于篇幅不在这里赘述，将在各个分类文档里列出。
:::
