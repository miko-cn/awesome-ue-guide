# 站点使命

> 一个完全免费、社区共建的虚幻引擎学习指南。

其实这件事的出发点很简单 —— 目前没有能够满足我的虚幻引擎指南。

有**非常非常多**的优质免费内容分散在互联网的各个角落，其中有官方维护的权威文档和教程，也有很多默默更新优质文章的博客站点，还有很多制作精良视频的UP主、油管主。

信息太多的问题就是令人目不暇接，我们需要一个**超级向导**，它可以告诉我哪些先看，哪些后看，哪些重点看，应该怎么看。

:D 不管如何，至少我需要这样的一个站点。

## 这里都有什么？

**站点主要会有以下内容**：

- 🔍 我们会收集网络上免费的优质资料；
- 🗃️ 对资料进行分类整理、编写引文介绍以及定期更新，方便检索和预览；
- ✒️ 编写面向不同阶段游戏开发者的学习路径推荐、精粹知识文章。
